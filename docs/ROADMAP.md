# CivilizationX Roadmap

## Now

- Keep CI green.
- Strengthen data validation.
- Make prerequisite and unlock relationships deterministic.
- Improve bottom-of-tree entries before expanding advanced branches.
- Add project identity, licensing decision, and contributor workflow.

## Next

- Add a Foundations view focused on primitive starting technologies.
- Add a pathfinder view for "how do we get from X to Y?"
- Add a raw-material atlas for Earth and space alternatives.
- Add risk gates beyond medicine for high-voltage, chemical, pressure, structural, radiation, and biological hazards.
- Add media fields for diagrams and videos with license tracking.

## Later

- Add contribution review tooling.
- Add expert review workflow.
- Add import tools for open-license media.
- Add printable field guides for selected technology chains.
- Add scenario modes:
  - Collapse recovery
  - Mars habitat
  - Orbital station
  - Island settlement
  - Low-resource rural clinic

## Open Questions

- Should the public name be CivilizationX, The Codex, or something else?
- What license should apply to code?
- What license should apply to curated content?
- Which expert domains should be recruited first?
- How detailed should build steps be for hazardous technologies?
