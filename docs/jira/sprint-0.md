# Sprint 0: Make The Project Manageable

Duration: 1 week

## Sprint Goal

Make CivilizationX manageable as a serious long-term project by establishing project direction, validation, CI, Foundations Mode, and a Jira backlog.

## Committed Scope

- Add project vision, roadmap, and master plan.
- Add Jira-ready backlog and workflow.
- Add CI.
- Strengthen data validation.
- Add unlock synchronization.
- Add Foundations Mode.
- Keep validation, lint, and build green.

## Stretch Scope

- Add entry maturity field design.
- Add hazard category schema design.
- Add pathfinder technical design.

## Exit Criteria

- The project has a clear first public story.
- The repo contains a backlog that can be imported into Jira.
- New contributors can understand where the project is going.
- Core checks pass locally.
